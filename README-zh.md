# 幂级数

通过幂级数展开来计算指数函数，正弦函数和余弦函数。随着计算项数的增加，结果会越来越接近实际值。

我用 Python 写了一个程序来计算幂级数。它在给定精度或位数的情况下，计算指数函数、正弦函数和余弦函数。

## 使用法

```bash
poetry install
poetry run py power-series.py
```
